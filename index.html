🌐 Uso Colaborativo — Sincronización con Sheets

☁️ Cómo funciona la sincronización institucional

Existe un único índice institucional guardado en Google Sheets que contiene todos los grados, cursos y estudiantes. El administrador lo carga una sola vez; cada docente lo consulta automáticamente al ingresar.

📥
Carga única

El administrador sube el JSON o Excel una sola vez por año/semestre.

👀
Visibilidad inmediata

Los docentes ven sus cursos al iniciar sesión, sin crear plantillas.

🚫
Sin duplicados

Todos editan el mismo índice — los cambios se sincronizan para todos.

⚙️ Panel de Administración

🔗 Conexión con Google Sheets (Apps Script)

La URL "de fábrica" (la que usan todos los docentes automáticamente, en cualquier navegador) está en el código (DEFAULT_GAS_URL). Este campo solo guarda una URL alternativa en ESTE navegador, útil para probar un despliegue nuevo antes de publicarlo para todos. Cuando confirmes que una URL nueva funciona, actualiza DEFAULT_GAS_URL en el código y vuelve a publicar el HTML — así todos los docentes quedan conectados sin tocar nada.

📄 Spreadsheet ID (solo si necesitas cambiarlo)

Si el Google Sheet actual se borró o cambió, pega aquí el ID del nuevo (la parte entre /d/ y /edit en su URL). Esto actualiza el script directamente — no necesitas volver a editar código.
